Manual de Usuario - Management Platform

1. Requisitos del Sistema

  • Haber instalado .NET 8 SDK.
  • Base de datos PostgreSQL activa.
  • Navegador Web (Chrome, Edge, etc).

2. Pasos de Instalación y Ejecución

  1. Crear la base de datos management_platform.
  2. Configurar la contraseña en la terminal: 
    dotnet user-secrets set "ConnectionStrings:DefaultConnection" "..." --project Platform.Web
  3. Ejecutar: dotnet run --project Platform.Web

3. Descripción de Funcionalidades

Proyectos

Permite crear proyectos en borrador, activarlos (si tienen tareas) y completarlos cuando todo esté finalizado.

Tareas

Asignación de tareas con prioridades y orden personalizado dentro de cada proyecto.
